## Admin Dashboard User Guide

This guide provides a comprehensive overview of the Admin Dashboard for the KSRCE AI Lab Management Platform. It is intended for administrators who will be managing the platform, its users, and its resources.

### 1. Accessing the Admin Dashboard

To access the Admin Dashboard, you must have an account with administrator privileges.

1.  **Log In**: Use your admin credentials to log in to the platform.
2.  **Navigate to Admin**: Once logged in, you will see an "Admin" link in the main navigation bar. Click on it to go to the dashboard.

### 2. Dashboard Overview

The main dashboard provides a high-level view of the platform's activity.

- **Key Metrics**: At a glance, you can see the total number of users, active bookings, total revenue, and GPU utilization.
- **Recent Activity**: A feed of the latest user registrations, bookings, and transactions.
- **Charts and Graphs**: Visual representations of user growth, revenue trends, and resource usage over time.

### 3. User Management

This section allows you to manage all users on the platform.

- **View Users**: A searchable and sortable list of all registered users.
- **User Details**: Click on a user to view their detailed profile, including their wallet balance, booking history, and activity log.
- **Edit User Roles**: You can change a user's role (e.g., from a student to a mentor).
- **Suspend or Delete Users**: If necessary, you can suspend a user's account temporarily or delete it permanently.

### 4. GPU Slot Management

Here, you can configure and manage the available GPU slots.

- **Add New Slots**: Define new GPU slots by specifying their name, hardware specifications, and hourly rate.
- **Edit Slots**: Modify the details of existing slots.
- **View Slot Availability**: See a real-time calendar of all booked and available slots.
- **Manual Booking**: As an admin, you can manually book a slot for a user.

### 5. Booking Management

This section provides tools to oversee all bookings.

- **View All Bookings**: A comprehensive list of all past, current, and future bookings.
- **Filter and Search**: Find specific bookings by user, date, or GPU slot.
- **Cancel Bookings**: You can cancel a user's booking if required. The system will automatically handle the refund to the user's wallet based on the cancellation policy.

### 6. Financial Management

Monitor and manage the financial aspects of the platform.

- **View Transactions**: A complete log of all financial transactions, including wallet recharges and booking payments.
- **Revenue Reports**: Generate reports on daily, weekly, or monthly revenue.
- **Payouts**: Manage payouts to mentors for their completed sessions.

### 7. Mentor Management

Oversee the mentors on the platform.

- **Approve Mentor Applications**: Review and approve applications from users who want to become mentors.
- **Set Mentor Rates**: Define the hourly rates for each mentor.
- **View Mentor Sessions**: See a schedule of all booked mentor sessions.

### 8. System Settings

Configure various platform-wide settings.

- **Payment Gateway**: Configure the API keys for Razorpay and other payment gateways.
- **Email Templates**: Customize the content of automated emails sent to users (e.g., welcome email, booking confirmation).
- **Platform Announcements**: Create and display announcements on the user dashboard.

This guide should provide you with all the information you need to effectively manage the KSRCE AI Lab Management Platform. For any further assistance, please refer to the technical documentation or contact the support team.
